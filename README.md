# João Pedro Dutra

**Desenvolvedor Full Stack · Analista de TI | Desenvolvimento**

Desenvolvo aplicações corporativas de ponta a ponta: levantamento de regra de negócio, banco de dados, API, interface web e mobile, automação e indicadores. Gosto de soluções que funcionam na vida real, com código tipado, documentado e em produção.

---

### O que faço hoje

- Aplicações corporativas web (React/TypeScript) e mobile (React Native/Expo) com back-end NestJS + Prisma + PostgreSQL
- Arquitetura offline-first para trabalho em campo (SQLite com sincronização), autenticação Azure AD e armazenamento em nuvem Azure
- Dashboards e indicadores com Power BI, DAX e Qlik Cloud, integrados a Postgres, Supabase e SharePoint
- Automação de processos com n8n, Docker e integrações via API e webhooks, incluindo atendimento via WhatsApp com agentes de IA

### Projetos em destaque

- **DutraGo**: plataforma SaaS multi-tenant de gestão de frotas e rastreamento de ativos
- **FlashBoyStrategy**: trading algorítmico com análise em Python e execução de baixa latência (<1ms) em C++ via pybind11, WebSocket da Binance e análise de sentimento com FinBERT
- **DirectTor**: app Android privacy-first com Tor onion v3, criptografia XChaCha20 e P2P mesh, sem persistência de dados
- **Mercado Smart**: análise de gastos via NF-e, com extração de itens no site do governo e alertas de variação de preço
- **Mãe-Radar**: rastreamento offline com Flutter e BLE
- **Capyidle**: jogo mobile idle publicado na Play Store, feito a pedido da minha sobrinha

Também desenvolvo sistemas corporativos sob NDA: plataforma de inspeções em campo, controle de portaria multiplataforma, gestão de processos (frota, almoxarifado e RH) e análise de perda operacional.
